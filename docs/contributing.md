# Contributing fitness-tracker

### This project is setup with linting and code format checking enabled, to help reduce unnecessary change merge request.

#### To automatically format all of the code with prettier . you can run

### npm run format

In order to merge code into `master`, we need to create a merge request (MR) in Github, which will be assigned to someone who can review it, before merging to master.

Before creating a merge request, you should run a check first, to make sure the build will pass.

You can do this by running:

### npm run precommit
